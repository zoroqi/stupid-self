# 440. 字典序的第K小数字

给定整数`n`和`k`，返回`[1, n]`中字典序第`k`小的数字。

```
示例 1:

输入: n = 13, k = 2
输出: 10
解释: 字典序的排列是 [1, 10, 11, 12, 13, 2, 3, 4, 5, 6, 7, 8, 9]，所以第二小的数字是 10。

示例 2:

输入: n = 1, k = 1
输出: 1
```

提示:

* `1 <= k <= n <= 10^9`

## 愚笨的思考

### 暴力方案 计划A

不考虑性能 haskell 很简单, 直接就可以 sort 获得对应元素就好了. sort 对字符串的默认排序就是 #字典序 , 所以只要把数字转成字符串就好了.

当然面对问题的范围, 这个方法不可行, 但 `n < 1000` 范围内其实还是可以的.
